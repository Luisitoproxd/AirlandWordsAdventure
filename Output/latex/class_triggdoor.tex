\doxysubsection{Detailed Description}
Cambia a una nueva escena cuando cualquier objeto entra en el área de colisión (trigger). 

Este script debe estar en un Game\+Object con un Collider2D marcado como trigger. Carga la escena cuyo nombre se indique en el Inspector.
